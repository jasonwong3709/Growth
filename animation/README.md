# Animation

Put **project-level** work here: treatments, shot lists, timing notes, and **small** exported ComfyUI workflow JSON files.

## Suggested layout

| Path | Use |
|------|-----|
| `projects/<name>/` | One folder per film / episode / experiment |
| `projects/<name>/brief.md` | Story and visual direction (your original writing) |
| `workflows/` | Exported ComfyUI graphs (`.json`), versioned in Git if small |
| `renders/` | **Heavy** exports (video, image sequences) — default is gitignored |

## Naming

Use a short slug, e.g. `projects/pilot-2026-03/`, `projects/short-winter-field/`.
